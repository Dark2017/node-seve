# node-serve

## 开始
```
建议使用淘宝镜像源（双倍快乐）安装依赖包：
cnpm i
运行命令：
npm start
```
## 技术栈
```
前端：vue、element-ui（组件库）
后端：nodejs（koa2）
数据库：mysql
```
